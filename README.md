# 합사 번수 계산기

뜨개질용 합사 번수 계산기입니다. 여러 가닥의 실 번수를 입력하면 합사된 실의 예상 최종 번수를 계산합니다.

## 기능

- 2가닥 이상 합사 계산
- 같은 번수 여러 가닥 계산
- 서로 다른 번수 여러 가닥 계산
- 계산식 표시
- 별도 설치 없이 브라우저에서 바로 실행

## 로컬에서 실행

`index.html` 파일을 브라우저로 열면 바로 사용할 수 있습니다.

## GitHub Pages로 배포하는 방법

### 1. GitHub에 새 저장소 만들기

예시 저장소 이름:

- `yarn-count-calculator`

### 2. 이 폴더의 파일 업로드

업로드할 파일:

- `index.html`
- `README.md`
- `.nojekyll`

### 3. GitHub Pages 켜기

GitHub 저장소에서:

1. `Settings`
2. `Pages`
3. `Build and deployment`에서 `Source`를 `Deploy from a branch`로 선택
4. `Branch`를 `main` / `/ (root)`로 선택
5. 저장

### 4. 배포 링크 확인

몇 분 뒤 아래 형태의 주소로 접속할 수 있습니다.

`https://사용자이름.github.io/저장소이름/`

예:

`https://yourname.github.io/yarn-count-calculator/`

## 참고

- 이 계산기는 번수 공식 기준의 참고용 계산기입니다.
- 실제 뜨개질에서는 재질, 꼬임, 탄성 차이 때문에 게이지 확인이 중요합니다.
