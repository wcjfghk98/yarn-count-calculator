# 스위치온 다이어트 컴패니언

아이폰 Safari에서 바로 열어 쓰기 좋은 개인용 스위치온 다이어트 웹앱입니다.
시작일을 넣으면 오늘이 몇 주차인지, 어떤 루틴을 지켜야 하는지, 식사 체크와 기록이 한 화면에 정리됩니다.

## 포함한 기능

- 4주 스위치온 다이어트 로드맵 요약
- 시작일 기준 오늘 단계 자동 계산
- 아침, 점심, 오후 간식, 저녁 개별 체크
- 권장 식사 시간대 표시
- 식사 시간 기록과 전날 대비 공복 시간 계산
- 물 2L, 수면, 영양제 등 일일 체크리스트
- 주간 운동 횟수 추적
- 허용음식과 피해야 할 음식 정리
- 영양제별 권장 타이밍 정리
- 월간 캘린더에서 기록 흐름 확인
- 몸무게, 허리둘레, 컨디션, 메모 저장
- 브라우저 `localStorage` 기반 개인 기록 유지
- `manifest.json` + `sw.js`를 통한 간단한 웹앱 형태 지원

## 파일 구성

- `index.html`: 메인 웹앱
- `manifest.json`: 홈 화면 추가용 웹앱 설정
- `sw.js`: 간단한 오프라인 캐시
- `.nojekyll`: GitHub Pages용

## 로컬에서 보기

그냥 `index.html`을 브라우저로 열어도 확인할 수 있습니다.
다만 서비스 워커와 홈 화면 추가 흐름까지 보려면 로컬 서버 또는 GitHub Pages 배포가 더 좋습니다.

## GitHub Pages 배포

현재 저장소가 GitHub 원격 저장소와 연결되어 있으므로 아래처럼 배포할 수 있습니다.

1. 변경사항 커밋
2. GitHub에 푸시
3. 저장소 `Settings > Pages`에서 `Deploy from a branch` 선택
4. `main` 브랜치의 `/ (root)` 배포

배포 후에는 아래 형태의 주소로 접속할 수 있습니다.

`https://wcjfghk98.github.io/yarn-count-calculator/`

## 아이폰에서 쓰는 방법

1. 배포된 링크를 Safari로 엽니다.
2. 하단 공유 버튼을 누릅니다.
3. `홈 화면에 추가`를 선택합니다.
4. 아이콘처럼 실행해서 기록합니다.
