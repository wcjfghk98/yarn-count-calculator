# 뜨개 게이지 환산기

도안 게이지와 내 스와치 게이지를 비교해서 완성 사이즈를 예측하고, 코수와 단수를 환산해 주는 단일 페이지 웹앱입니다.

## 포함된 기능

- 도안 대비 실제 완성 크기 비율 계산
- 예상 폭과 길이 계산
- 코수 환산
- 단수 환산
- 원하는 cm 기준 코수와 단수 계산
- 브라우저에서 바로 실행되는 단일 `index.html`

## 사용 방법

1. `index.html`을 브라우저에서 엽니다.
2. 도안 코 게이지와 단 게이지를 입력합니다.
3. 내 스와치의 코 게이지와 단 게이지를 입력합니다.
4. 필요하면 도안 완성 치수, 도안 코수/단수, 원하는 폭/길이를 함께 입력합니다.
5. 계산 결과가 오른쪽 패널에 즉시 반영됩니다.

## 계산 공식

- 가로 비율 = 도안 코 게이지 ÷ 내 코 게이지
- 세로 비율 = 도안 단 게이지 ÷ 내 단 게이지
- 새 코수 = 기존 코수 × (내 코 게이지 ÷ 도안 코 게이지)
- 새 단수 = 기존 단수 × (내 단 게이지 ÷ 도안 단 게이지)
- 필요 코수 = (내 코 게이지 ÷ 10) × 원하는 폭(cm)
- 필요 단수 = (내 단 게이지 ÷ 10) × 원하는 길이(cm)

## 메모

- 코 게이지는 가로 크기에 영향을 줍니다.
- 단 게이지는 세로 길이에 영향을 줍니다.
- 무늬, 합사, 세탁 후 변화가 있으면 같은 조건으로 스와치를 다시 재는 편이 정확합니다.
