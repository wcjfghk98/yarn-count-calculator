합사 번수 계산기

서로 다른 실을 여러 가닥 합쳤을 때의 최종 번수를 계산합니다. 공식은 1 / 최종 번수 = 1/N1 + 1/N2 + ... 를 사용하며, 결과는 참고용이므로 실제 뜨개 전에는 게이지 스와치를 함께 확인하는 것이 좋아요.

실 정보 입력

각 가닥의 번수를 입력하세요. 같은 실을 3합, 4합으로 쓰는 경우도 한 줄씩 추가하면 됩니다.

계산 결과

예상 최종 번수

-

적용 공식

값을 입력하면 계산식이 여기에 표시됩니다.

실전 메모

  • 숫자가 작을수록 더 굵은 실입니다.
  • 같은 번수 여러 가닥은 사실상 `번수 ÷ 가닥 수`와 같은 결과가 납니다.
  • 재질, 꼬임, 탄성 차이 때문에 최종 느낌은 게이지가 가장 중요합니다.
